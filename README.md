## Para rodar

É necessário instalação previa do nodejs.

Instalando dependências:

```bash
npm install
```

Rodando front-end:

```bash
npm run dev
```

Rodando back-end:

```bash
npm run server
```

### Buildar para produção:

Etapa necessária para se observar a performance da geração estática de sites

```bash
npm run build
```

```bash
npm run start
```
